Club Records

Official club records for various age groups and distances

Below are the club records for various age groups over a number of classic race distances. To qualify for a place on the chart, the time should have been set in a licensed UK Athletics road race or parkrun event.

The current charts have been compiled using information available from only the last few years. If any member can provide details to help complete the gaps, or can correct/update what is already there, please contact Jon Vaile.
